# County Details Management - Quick Reference Guide

## Quick Access

### Open County Details
1. Go to **Counties** page from sidebar
2. Click on any county row, OR click the **"View Details"** button
3. Modal opens with complete county information

## What You'll See

### County Information Section
- County name and URL slug
- Region and capital city
- Motto/tagline
- Established year
- Current status badge

### Statistics Section (Real-time)
- **Contributors** - County admin users count
- **Content Items** - Total published content
- **Opportunities** - Job/opportunity listings
- **Events** - Upcoming/past events

### Action Buttons

| Button | Purpose |
|--------|---------|
| **Edit County** | Modify county information and save |
| **View Public Site** | Open county page in new tab |
| **Manage Contributors** | Assign/manage county admins |
| **Manage Content** | Upload/publish county content |
| **Manage Opportunities** | Create/edit job opportunities |
| **Manage Events** | Schedule and manage events |

## Common Tasks

### Edit County Information
```
1. Click "View Details" on county
2. Click "Edit County" button
3. Update any fields (name, region, capital, motto, year, status)
4. Click "Save Changes"
```

### Check County Statistics
```
1. Click county row
2. View colored stat cards showing:
   - Blue: Contributors
   - Green: Content items
   - Purple: Opportunities
   - Orange: Events
```

### Visit County Public Page
```
1. Open county details
2. Click "View Public Site"
3. County page opens in new tab
```

### Manage County Resources
```
1. Open county details
2. Click relevant action button:
   - Contributors → Manage admins
   - Content → Upload/edit content
   - Opportunities → Create jobs
   - Events → Schedule events
```

## Tips & Tricks

✅ **Click the row** - Faster than clicking the button  
✅ **Status changes** - Use Edit to change live/draft/coming soon status  
✅ **Real-time stats** - Numbers update when you manage resources  
✅ **Mobile friendly** - All features work on phones and tablets  
✅ **Easy editing** - All info in one edit form  

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Modal won't open | Ensure you're logged in as Super Admin |
| Stats show 0 | County may be new with no content yet |
| Edit button grayed out | Check your permissions |
| Changes not saving | Check internet connection and try again |

## Data Fields

### Basic Information
- **County Name** - Required, unique identifier
- **Slug** - URL-friendly name (auto-generated or custom)
- **Region** - Geographic region
- **Capital** - Main city/capital
- **Motto** - County tagline or motto

### Administrative
- **Status** - Live, Coming Soon, or Draft
- **Established Year** - When county was founded
- **County Admin** - Assigned administrator
- **Created/Updated** - Timestamp information

## Notes

📌 Changes save immediately to database  
📌 All SuperAdmins can edit any county  
📌 Deleted data cannot be recovered  
📌 Timestamps show in your local timezone  
📌 Status affects visibility on public site
